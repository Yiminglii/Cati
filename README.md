# 🐱 Cati - 猫格测试

**测测你和猫的契合度有多高 💕**

你的 MBTI ✕ 你家猫的猫型人格 = 一份专属的人猫契合报告

![Cati Preview](cati.html)

## ✨ 项目介绍

Cati 是一款有趣的**人猫契合度测试**应用，通过 MBTI 人格类型理论结合猫咪行为学，帮你了解你家猫咪的「猫型人格」，并计算你与它的默契程度。

## 🎮 功能特性

### 📱 H5 网页版 (`cati.html`)
- 🎨 精美粉色系 UI，流畅动画效果（爪印、彩纸庆祝）
- 🧠 输入你的 MBTI 四维性格
- 📝 **24 道全原创猫咪题目**：
  - 12 道「猫猫第一人称」视角
  - 12 道「铲屎官观察视角」
- 🔍 覆盖全部 4 个维度：E/I、S/N、T/F、J/P
- 🎯 **16 种猫型人格结果**（每种都有独特描述 + 口号 + 特质标签）
- ❤️ 契合度计算与等级评定（42%~98%）
- 📤 一键分享测试结果

### 📲 微信小程序版 (`cati-mini/`)
- 完整的小程序项目结构
- 原生 WXML/WXSS 开发
- 支持微信分享给好友和朋友圈
- 震动反馈等原生交互体验

## 🚀 快速开始

### H5 版
直接在浏览器中打开 `cati.html` 即可使用，无需任何依赖。

### 小程序版
1. 用 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 打开 `cati-mini` 目录
2. 配置 AppID（或使用测试号）
3. 点击编译即可预览

## 🛠 技术栈

- **H5 版**: HTML5 + CSS3 + Vanilla JavaScript（单文件，零依赖）
- **小程序版**: 微信小程序原生开发（WXML + WXSS + JS）
